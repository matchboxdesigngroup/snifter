    get_custom_meta_fields() : array

    By returning an empty array in this method, we're telling the class to not to do anything will custom meta (e.g. meta boxes, and saving meta etc...) The overridden method should return an array like.. return array( array( 'label' => 'Title/Position', 'desc' => '', 'id' => $prefix.'Title', 'type' => 'text', ), array( 'label' => 'Quote', 'desc' => '', 'id' => $prefix.'Quote', 'type' => 'textarea', ) );

    Returns

    Retrieves custom post meta.

    get_post_meta(integer $post_id = null, string $key = null) : string | array

    global $my_post_type;

    // Retrieves all the current posts meta. $all_post_meta = $my_post_type->get_post_meta(); extract( $all_post_meta );

    // Retrieves a single post meta value. $single_post_meta = $my_post_type->get_post_meta( get_the_id(), 'meta_key'); extract( $single_post_meta );

    Parameters

    $post_id

    integer

    Optional post id, defaults to current post.

    $key

    string

    Optional meta key minus the post type, to return one meta value instead of all of them.

    Returns

    stringarraySpecific post meta value | All all post meta values.
